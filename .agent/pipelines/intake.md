# Intake Stage

## Goal
Convert user intent into a scoped, constraint-aware work request.

## Checklist
1. Identify target output (analysis, changes, design, or mixed).
2. Determine route: QA, Change, or Proposal.
3. Resolve effective instruction chain from `AGENTS.md`.
4. Select minimal relevant skills from `SKILLS.md`.
5. Produce concise intake summary for planning.
