# Team Planning Checklist Template

Use this template for management and cross-team planning sessions.

## Required preflight gates

- [ ] **SoT freshness gate passed before sprint-planning**
  - [ ] Reviewed all source-of-truth links in `docs/operations/execution_index.md`
  - [ ] Verified each status section has a `Date (UTC)` update within 7 days
  - [ ] If stale entries were found, planning was blocked and owners refreshed entries first
  - [ ] Gate result logged using `docs/operations/runbooks/sprint_planning_source_of_truth_gate.md`

## Sprint planning readiness

- [ ] Scope and priorities confirmed
- [ ] Dependencies and blockers reviewed
- [ ] Capacity assumptions documented
- [ ] Delivery risks and mitigations captured

## Session outputs

- [ ] Sprint objective recorded
- [ ] Story/task sequencing recorded
- [ ] Owners and due dates assigned
- [ ] Follow-up actions tracked
