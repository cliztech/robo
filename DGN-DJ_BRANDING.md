# DGN-DJ Branding Notes

## Logo
The official DGN-DJ logo has been provided as a neon-green gorilla wearing headphones on a black background.

## Usage guidance
- Keep the black background intact for strongest contrast.
- Preserve the neon-green color treatment and glow effect.
- Avoid recoloring or stretching the mark.
- Keep clear spacing around the logo so the headphone silhouette remains readable.

## Asset handoff note
This repository currently stores distribution/configuration artifacts only. If you want the raw logo image committed here, add the image file (PNG/JPG/SVG) to the repo and reference it from this document.
